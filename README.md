## 📌 Project Overview

This project is a Machine Learning-based Network Intrusion Detection System developed using the CIC-IDS2017 dataset.

The system classifies network traffic as:

- ✅ BENIGN
- 🚨 DDoS Attack

---

## 🤖 Models Used

- Logistic Regression
- Decision Tree
- Random Forest

---

## 📊 Results

### Logistic Regression
Accuracy: ~96%

### Random Forest
Accuracy: ~100%
